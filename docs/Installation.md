\# Installation Guide



\## Requirements



\- Windows 10/11

\- Windows Server 2019+

\- Node.js 24+

\- PowerShell 5.1+

\- Git



\## Clone Repository



```bash

git clone https://github.com/saravanakumarofficial2003-collab/RPP-Galaxy.git

```



\## Install Packages



```bash

cd Server

npm install

```



\## Start Server



```bash

node server.mjs

```



\## Open Dashboard



```

http://localhost:8090

```



or



```

http://YOUR-IP:8090

```
